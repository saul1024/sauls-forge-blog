## 什么是 Prompt Engineering？

Prompt Engineering（提示工程）是一门与大语言模型（LLM）有效沟通的技术。简单来说，就是学会如何向 AI 提出好问题，以获得更准确、更有用的回答。

## 为什么重要？

在 AI 时代，能否有效地与 AI 协作已经成为一项核心技能。好的 Prompt 和差的 Prompt 之间的差距，可能是天壤之别。

## 基本技巧

### 1. 明确角色

```
你是一个资深的 Python 开发工程师，请帮我...
```

### 2. 提供上下文

```
背景：我正在开发一个博客系统，使用 Hexo 框架。
问题：如何实现文章的自动分类功能？
```

### 3. 指定输出格式

```
请以 Markdown 表格的形式对比以下三个框架的优缺点：
- React
- Vue
- Angular
```

## 进阶策略

- **Chain of Thought**：引导 AI 逐步推理
- **Few-shot Learning**：给出几个示例
- **Self-consistency**：多次生成取最优

## 小结

Prompt Engineering 不仅仅是技术，更是一种思维方式。掌握它，你就能在 AI 时代如鱼得水。
